# Grand Bazaar — Department Store Frontend

A full-featured department store e-commerce frontend built with React + Vite.

## Tech Stack
- **Framework**: React 19 + Vite
- **Routing**: React Router DOM v7
- **State**: React Context API
- **HTTP**: Axios
- **Animations**: Framer Motion
- **Icons**: React Icons
- **Notifications**: React Toastify

## Features
- Multi-department shopping (Electronics, Home & Living, Fashion, Groceries, Sports, Beauty)
- Product filtering by department, brand, price range, size, color
- Cart with coupon support
- Wishlist
- User authentication (login, register, forgot password)
- Order management with invoice generation
- Admin panel: manage departments, brands, products, orders, users, coupons, banners, reviews
- Dark mode support

## Setup
```bash
npm install
npm run dev
```
